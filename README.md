# X6-Drinks

A modern web project for the X6 Energy Drink brand.

## Codebase Overview

- **/components**: Reusable React components (UI, primitives)
- **/src**: Source files, including main entry, additional styles
- **/styles**: Global and utility CSS (Tailwind CSS)
- **index.html**: Main HTML entry for Vite/React

## Main Technologies

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Radix UI](https://www.radix-ui.com/) (for primitives like scroll area)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.
